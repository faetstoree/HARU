# HARU — 留学生のための AI 生活ガイド

**HARU** は、来日したばかりの留学生のために設計された AI コンパニオンです。  
入国から生活の立ち上げまで、あなたの状況に合わせて必要な手続きと生活準備をステップごとにサポートします。

> [繁體中文](./README_ZH.md) ｜ [English](./README_EN.md) ｜ 日本語

---

## HARU でできること

来日直後は、窓口ごとに期限や前提条件が異なる手続きが山積みです。HARU はそれらを整理し、**あなた専用のパーソナライズされたロードマップ**として提示しながら、いつでも質問に答えます。

### 🗺️ パーソナライズされた定住ロードマップ

学校・住居形態・来日日・アルバイト予定などをもとに、順序立てられたタスクリストを自動生成します。AI がタスクごとに、あなたの状況に合った説明・手順・アドバイスを作成します。

対応している手続きの例：
- 区役所での住所登録
- 国民健康保険の加入手続き
- SIM カード・携帯番号の取得
- 銀行口座の開設
- 学校への入学手続き
- 資格外活動許可（アルバイト許可）の申請
- …その他、あなたのプロフィールに応じて動的に追加されるタスク

### 🤖 AI チャットアシスタント「Haru」

いつでも質問でき、すぐに回答が得られます。Haru にできること：
- 近くの区役所・銀行・携帯ショップの**リアルタイム営業時間**を検索
- 会話の中に **Google マップ**のナビゲーションを埋め込み
- 現在のタスク進捗に基づき、次にすべきことを能動的に提案
- **音声入力・読み上げ**（TTS / STT）に対応

### 📋 診断クイズ

何から始めればいいかわからない場合は、クイズに答えるだけで OK。Haru があなたの状況を分析し、優先順位を示した AI 診断レポートを生成します。

### 📄 行動計画 PDF の書き出し

現在のタスクリストと手順を PDF に書き出し。窓口に持参する際に役立ちます。

### 🌐 3 言語対応

インターフェースと AI の回答が完全に対応：
- 繁體中文（繁体字中国語）
- English（英語）
- 日本語

## はじめ方

### 1. HARU を開く

ブラウザで HARU の URL を開きます（管理者またはデプロイ先から提供されたリンク）。

### 2. オンボーディングを完了する

初回アクセス時に、基本情報を入力します：

| 項目 | 説明 |
|---|---|
| 学校名 | 在籍している日本の学校名 |
| 来日日 | 日本に入国した日付 |
| 居住地域 | 都道府県または市区町村名 |
| 日本語レベル | ゼロ / 初級 / 中級 / 上級 |
| 学校の種類 | 語学学校 / 大学 / 専門学校 / 高校 |
| 住居形態 | 寮 / 自己賃貸 / 未定 |
| アルバイト予定 | しない / する予定 / あとで決める |

**すでに完了している手続き**（在留カード受け取り済み、空港で SIM を契約済みなど）にチェックを入れると、HARU が自動的にそのステップをスキップします。

### 3. ロードマップを確認する

オンボーディング完了後、AI があなた専用のロードマップを即座に生成します。タスクはフェーズごとに整理され、以下の状態で表示されます：
- ✅ 完了済みのタスク
- ▶️ 今すぐ取りかかれるタスク
- 🔒 前提タスクが未完了のため、まだできないタスク

任意のタスクをクリックすると、詳細説明・必要書類・公式リンク・AI 生成の個人化手順が確認できます。

### 4. Haru と話す

画面右下のチャットパネルを開いて、文字または音声でなんでも質問してみましょう：

- 「近くの区役所は何時に開きますか？」
- 「国民健康保険の加入に必要な書類は？」
- 「今すぐ何をすべきですか？」

### ロードマップビュー

```
┌─────────────────────────────────────────────────┐
│  フェーズ1  │  フェーズ2  │  フェーズ3  │  …    │  ← フェーズタブ
├─────────────────────────────────────────────────┤
│                                                 │
│  [✅ 住所登録] → [▶️ 健保加入] → [🔒 銀行口座] │  ← タスクフロー
│                                                 │
├─────────────────────────────────────────────────┤
│  タスク詳細パネル                               │
│  • 説明 / 必要書類 / 公式リンク                │
│  • AI 個人化ステップ                           │
└─────────────────────────────────────────────────┘
```

### タスクの状態

| 状態 | 意味 |
|---|---|
| ✅ 完了 | 完了済みとしてマーク済み |
| ▶️ 進行中 | 今すぐ取りかかれる |
| 🔒 ロック中 | 前提タスクが未完了 |
| ⏭️ スキップ | あなたの状況では不要（自動判定） |

### 分岐の選択

一部のタスクは選択によって分岐します。たとえば「自己賃貸」の流れは「寮」とは異なります。HARU は適切なタイミングで確認を求め、その後のタスクリストを自動的に調整します。

## 設定と API キー

HARU は Google Gemini API を利用して AI 機能を動かしています。キーの設定方法は 2 つあります：

### 個人キー（推奨）

HARU 画面 → **プロフィール（設定）** → **API キー** に自分の Gemini API キーを入力します。  
このキーはあなたのデバイスにのみ保存され、**他のユーザーには影響しません**。

> 無料キーの取得：[https://aistudio.google.com/apikey](https://aistudio.google.com/apikey)

### サーバーキー

管理者がサーバー側に共有キーを設定している場合、追加設定なしでそのまま使用できます。

### キーがない場合

HARU は **モックモード**で動作し、あらかじめ用意されたデモ回答を使用します。インターフェースを試すことはできますが、AI 個人化機能は有効になりません。

## よくある質問

**Q：データはどこに保存されますか？**  
A：プロフィールとタスクの進捗は Google Cloud Firestore に保存されます。デバイス ID で識別されるため、アカウント登録は不要です。

**Q：ロードマップは自分の状況に合わせて変わりますか？**  
A：はい。学校の種類・住居形態・アルバイト予定などのプロフィール情報に応じて、表示されるタスクと順序が変わります。プロフィールを更新すると、AI ロードマップが自動的に再生成されます。

**Q：音声機能を使うために特別な設定は必要ですか？**  
A：音声機能はサーバー側で管理者が有効化します。使用できない場合は、サーバーに関連する Google Cloud サービスが設定されていない可能性があります。

**Q：スマートフォンでも使えますか？**  
A：はい。HARU はレスポンシブデザインに対応しており、スマートフォンやタブレットのブラウザでも利用できます。

**Q：Haru が提供する情報は正確ですか？**  
A：Haru の知識ベースは信頼できる公式情報源から構築されており、AI チャットには誤情報を防ぐため検証済みの事実が注入されています。ただし、日本の行政手続きは随時変更される可能性があるため、重要な判断は必ず関連する公式サイトで最新情報を確認してください。
