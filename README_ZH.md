# HARU — 留日新生 AI 生活嚮導

**HARU** 是專為剛抵達日本的留學生設計的 AI 伴侶。  
從入境到安頓，HARU 會依照你的個人狀況，逐步帶你完成所有必要的行政手續與生活設置。

> 繁體中文 ｜ [English](./README_EN.md) ｜ [日本語](./README_JA.md)

---

## HARU 能幫你做什麼？

剛到日本，有一大堆手續得跑，時間窗口又各不相同。HARU 把這些整理成一份**專屬你的個人化路線圖**，並隨時回答你的問題。

### 🗺️ 個人化入住路線圖

根據你的學校、居住形式、抵台日期、打工計畫等，自動生成一份有順序的任務清單。AI 還會幫每個任務寫出量身訂做的說明、步驟與小提醒。

涵蓋手續包括：
- 區役所住居登錄
- 國民健康保險申辦
- SIM 卡 / 手機門號
- 銀行帳戶開設
- 學校報到手續
- 資格外活動許可（打工簽）
- …以及更多依個人情況生成的任務

### 🤖 AI 聊天助理「Haru」

隨時問、即時答。Haru 能：
- 查詢附近區役所、銀行、手機店的**即時營業時間**
- 在對話中嵌入 **Google Maps** 導覽
- 根據你目前的任務進度，主動提供下一步建議
- 支援**語音輸入與語音朗讀**（TTS / STT）

### 📋 診斷問卷

不確定自己現在應該先做什麼？透過快速問卷，Haru 會分析你的狀況，生成一份 AI 診斷報告，告訴你優先級。

### 📄 行動計畫 PDF 匯出

把你目前的任務清單和步驟匯出成 PDF，帶去辦手續時可以對照使用。

### 🌐 三語支援

介面與 AI 回應完整支援：
- 繁體中文
- English
- 日本語

## 快速開始

### 1. 進入 HARU

用瀏覽器開啟 HARU 網址（或由管理員提供的部署連結）。

### 2. 完成 Onboarding

首次進入時，填寫基本資料：

| 欄位 | 說明 |
|---|---|
| 學校名稱 | 你就讀的日本學校 |
| 抵達日期 | 入境日本的日期 |
| 居住地區 | 都道府縣或城市名稱 |
| 日語程度 | 零基礎 / 初級 / 中級 / 進階 |
| 學校類型 | 語言學校 / 大學 / 専門学校 / 高中 |
| 居住形式 | 宿舍 / 自租房 / 未定 |
| 打工計畫 | 不打算 / 計畫打工 / 之後再說 |

勾選你**已經完成**的事項（例如已拿到在留卡、已在機場辦好 SIM），HARU 會自動跳過這些步驟。

### 3. 查看你的路線圖

Onboarding 完成後，AI 會立即生成你的個人化路線圖。路線圖按階段排列，顯示：
- ✅ 已完成的任務
- ▶️ 現在可以進行的任務
- 🔒 需要先完成前置任務才能解鎖的任務

點擊任何任務，可以看到詳細說明、所需文件、官方連結，以及 AI 生成的個人化步驟。

### 4. 和 Haru 聊天

畫面右下角可以開啟聊天視窗，直接用文字（或語音）問任何問題，例如：

- 「附近的區役所幾點開？」
- 「辦國民健康保險需要帶什麼？」
- 「我現在應該先做哪件事？」

## 介面說明

### 路線圖視圖

```
┌─────────────────────────────────────────────────┐
│  Phase 1  │  Phase 2  │  Phase 3  │  Phase 4    │  ← 階段標籤
├─────────────────────────────────────────────────┤
│                                                 │
│   [✅ 住居登錄] → [▶️ 健保申辦] → [🔒 開銀行]   │  ← 任務流程圖
│                                                 │
├─────────────────────────────────────────────────┤
│  任務詳情面板                                   │
│  • 說明 / 所需文件 / 官方連結                   │
│  • AI 個人化步驟                               │
└─────────────────────────────────────────────────┘
```

### 任務狀態

| 狀態 | 說明 |
|---|---|
| ✅ 已完成 | 任務已標記為完成 |
| ▶️ 進行中 | 目前可以執行 |
| 🔒 鎖定 | 需要先完成前置任務 |
| ⏭️ 跳過 | 依你的狀況不適用（自動判斷） |

### 分支選擇

部分任務會依你的選擇產生分支，例如「自租房」流程會和「宿舍」流程不同。HARU 會在適當時機請你確認選擇，並調整後續任務。

## 設定與 API Key

HARU 使用 Google Gemini API 驅動 AI 功能。Key 有兩種設定方式：

### 個人 Key（推薦）

在 HARU 介面 → **Profile（設定）** → **API Key** 輸入你自己的 Gemini API Key。  
此 Key 只儲存在你的裝置，**不影響其他用戶**。

> 取得免費 Key：[https://aistudio.google.com/apikey](https://aistudio.google.com/apikey)

### 伺服器 Key

若管理員已在伺服器設定了共用 Key，你不需要額外設定，直接使用即可。

### 沒有 Key 時

HARU 會進入 **Mock 模式**，使用預先準備好的示範回應，讓你熟悉介面，但 AI 個人化功能不會啟用。

## 常見問題

**Q：我的資料存在哪裡？**  
A：你的 Profile 和任務進度儲存在 Google Cloud Firestore，以裝置識別碼區分，不需要帳號登入。

**Q：路線圖會根據我的狀況調整嗎？**  
A：會。你填寫的學校類型、居住形式、打工計畫等，都會影響出現的任務清單和順序。Profile 更新後路線圖會自動重新生成。

**Q：語音功能需要特別設定嗎？**  
A：語音功能由管理員在伺服器端啟用，若目前無法使用，可能是伺服器尚未設定相關服務。

**Q：我可以在手機上用嗎？**  
A：可以。HARU 的介面是響應式設計，支援手機和平板瀏覽器。

**Q：Haru 提供的資訊準確嗎？**  
A：Haru 的知識庫來自可信的官方資源，且 AI 聊天會注入相關知識以降低誤答率。但日本政府手續可能隨時更新，重要決定前建議再到官方網站確認。
